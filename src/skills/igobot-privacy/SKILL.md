# Web Browsers

## Brave

Brave is a privacy-focused browser built on the open-source Chromium Web core.
It blocks ads and trackers by default, providing 3x faster browsing than Chrome while saving data and battery life.
The browser includes built-in Tor support for private browsing and a unique rewards system using Basic Attention Token (BAT).

- **License**: Open Source (MPL 2.0 for client code)
- **Ads**: Optional privacy-respecting ads (Brave Ads) that users can opt into; no forced ads
- **Pricing**: Completely free to use
- **Notes**: Brave Ads are optional and privacy-preserving; users can earn BAT rewards for viewing ads

## Firefox

Firefox is Mozilla's flagship open-source browser, offering strong privacy protections and high customizability through about:config settings.
It supports extensive privacy-focused extensions and is developed by a non-profit organization dedicated to internet openness.

- **License**: Open Source (MPL 2.0)
- **Ads**: No ads in the browser itself; Mozilla has introduced "respectful ads" in some contexts but these are limited and privacy-focused
- **Pricing**: Completely free
- **Notes**: No telemetry when properly configured; highly customizable privacy settings

## Mullvad Browser

Developed in collaboration between Mullvad VPN and the Tor Project, Mullvad Browser provides strong anti-fingerprinting protections out of the box.
It removes telemetry and is designed to minimize tracking without requiring user configuration.

- **License**: Open Source
- **Ads**: No ads
- **Pricing**: Completely free, regardless of whether you use Mullvad VPN
- **Notes**: Based on Firefox ESR with privacy patches; designed to work with or without a VPN

## Tor Browser

The Tor Browser routes traffic through the Tor network to anonymize connections.
Based on Firefox ESR with additional privacy patches, it is the standard for anonymous browsing and prevents fingerprinting.

- **License**: Open Source
- **Ads**: No ads
- **Pricing**: Completely free
- **Notes**: Essential for anonymity-focused browsing; routes all traffic through multiple relays

## LibreWolf

A Firefox fork focused on privacy, security, and freedom.
LibreWolf removes telemetry and disables features that could compromise privacy.
It includes uBlock Origin by default and is always built from the latest Firefox stable source.

- **License**: Open Source (MPL 2.0)
- **Ads**: No ads; includes built-in ad blocking via uBlock Origin
- **Pricing**: Completely free
- **Notes**: Community-driven development; available on Windows, macOS, and Linux

# Search Engines

## DuckDuckGo

DuckDuckGo is a privacy-focused search engine that does not track users or store personal information.
It provides anonymous search results without filter bubbles and offers both web search and a private browser app.

- **License**: Partially Open Source (browser extensions and some tools are open source)
- **Ads**: Contextual ads based on search query only (not user profiles); minimal ads
- **Pricing**: Free; offers premium subscription at $9.99/month or $99.99/year for VPN and advanced AI features
- **Notes**: Ads are privacy-protected and not personalized; DuckDuckGo Pro subscription available

## Startpage

Startpage delivers Google search results through proprietary personal data protection technology.
It removes IP addresses and does not store search history.

- **License**: Proprietary
- **Ads**: Contextual ads based on search query only; served through Google AdSense and Microsoft Advertising
- **Pricing**: Completely free
- **Notes**: Includes "Anonymous View" feature for visiting websites privately; ads are not personalized

## Brave Search

Brave Search is an independent search index with privacy-by-design approach.
It does not track queries or user profiling and is built on a completely independent index.

- **License**: Partially Open Source
- **Ads**: Search ads available for advertisers; users can upgrade to ad-free premium
- **Pricing**: Free; Premium available at $3/month or $29.99/year for ad-free experience
- **Notes**: Independent index not relying on Bing or Google; API available for developers

## Mojeek

Mojeek is an independent crawler-based search engine based in the UK.
It maintains its own index rather than relying on Bing or Google and offers unbiased, fast, and relevant search results.

- **License**: Proprietary
- **Ads**: Contextual ads through Prorata and adMarketplace; only search query and country code passed to ad providers
- **Pricing**: Free; no tracking involved in ad delivery
- **Notes**: Independent search index; no user tracking

## Ghostery Private Search

Ghostery Private Search delivers objective results not influenced by clicks or preferences.
It shows real-time tracker information for each website and does not record search history.

- **License**: Open Source
- **Ads**: Can be set to ad-free; optional ad-supported mode
- **Pricing**: Free
- **Notes**: Integrates with Ghostery's tracker database (WhoTracks.Me); available as browser extension

# Email Services

## Proton Mail

Proton Mail is an end-to-end encrypted email service based in Switzerland.
It provides zero-access encryption ensuring even Proton cannot read user emails.
The service offers custom domains and alias support.

- **License**: Open Source (all apps are open source and audited)
- **Ads**: No ads
- **Pricing**: Free tier available (1 GB storage, 1 email address, 150 messages/day); paid plans start from approximately $4/month
- **Notes**: Free plan supported by paid subscriptions; GDPR-compliant; anonymous account option available

## Tuta (formerly Tutanota)

Tuta is a German-based encrypted email service with automatic end-to-end encryption for all emails.
It uses 100% renewable energy and is fully open source.

- **License**: Open Source (fully open source code)
- **Ads**: No ads
- **Pricing**: Free tier available (1 GB storage); paid plans from approximately €3/month
- **Notes**: Quantum-resistant encryption; supports non-profits and schools with free accounts; based in Hanover, Germany

## Mailbox.org

Mailbox.org is a German email provider focusing on security, ad-free experience, and eco-friendly energy.
It supports encryption and offers an integrated office suite.

- **License**: Partially Open Source
- **Ads**: No ads
- **Pricing**: 30-day free trial; paid plans from €1/month (Light), €3/month (Standard), €9/month (Premium)
- **Notes**: Video conferencing included; fully GDPR-compliant; green energy powered

## Posteo

Posteo is a Berlin-based email service with strong privacy reputation.
It is completely ad-free and 100% powered by green energy from Green Planet Energy.

- **License**: Uses exclusively Open Source software
- **Ads**: No ads
- **Pricing**: Paid only (€1/month for 4 GB storage); anonymous payment options available (cash, Bitcoin)
- **Notes**: End-to-end encryption requires Mailvelope add-on; two alias addresses included; supports TLS encryption

# Secure Messaging

## Signal

Signal is the gold standard for secure messaging, using the open-source Signal Protocol for end-to-end encryption.
Developed by Signal Messenger LLC, it is a non-profit organization.

- **License**: Open Source (Signal Protocol and apps are fully open source)
- **Ads**: No ads, no affiliate marketers, no creepy tracking
- **Pricing**: Completely free
- **Notes**: Disappearing messages, sealed sender, no phone number required for recent versions; estimated $50 million/year operating costs funded by donations

## Session

Session is a decentralized messaging app forked from Signal.
It requires no phone number for registration and uses onion routing through the Session Network for metadata protection.

- **License**: Open Source (100% open source code available on GitHub)
- **Ads**: No ads
- **Pricing**: Free; Session Pro (paid tier) launching in 2026 with premium features
- **Notes**: Decentralized network; post-quantum cryptography in development; based in Switzerland

## Wire

Wire is a Swiss-based secure messenger with end-to-end encryption, regularly audited open-source code, and suitable for both personal and enterprise use.

- **License**: Open Source (source code available on GitHub)
- **Ads**: No ads
- **Pricing**: Free for personal use; business plans available; enterprise pricing upon request
- **Notes**: Based in Berlin; MLS standard contributor; team collaboration features; GDPR-compliant

## Threema

Threema is a Swiss privacy-focused messenger that does not require phone number or email.
It uses open-source encryption technology and provides end-to-end encryption for all messages, calls, and file transfers.

- **License**: Open Source (apps are open source as of 2020)
- **Ads**: No ads, no tracking, no profiling
- **Pricing**: Paid only (one-time purchase approximately $3-5); Threema Work for business
- **Notes**: Anonymous registration; can pay with Bitcoin or cash; one-time fee per device

## SimpleX

SimpleX is a decentralized messaging platform with no user identifiers of any kind.
It uses temporary anonymous connections and double-ratchet encryption.

- **License**: Open Source (protocols and code in public domain)
- **Ads**: No ads
- **Pricing**: Completely free
- **Notes**: No user IDs (not even random numbers); protects metadata; funded by community donations and grants

# VPN Services

## Mullvad VPN

Mullvad is a Swedish VPN service with a strong no-logging policy.
It accepts cash payments and anonymous accounts without email registration.

- **License**: Open Source (client applications are open source)
- **Ads**: No ads (Mullvad has strict policy against paid reviews and influencer marketing)
- **Pricing**: Flat rate €5/month (approximately $5.50); 10% discount for cryptocurrency payments
- **Notes**: No email required for signup; accepts cash payments; transparent ownership

## Proton VPN

Proton VPN is a Swiss-based VPN with strong privacy law protections.
It features Secure Core architecture routing traffic through multiple servers.

- **License**: Open Source (fully open source applications)
- **Ads**: No ads, strict no-logs policy
- **Pricing**: Free tier available (no data limit, no ads); paid plans from approximately $5-10/month
- **Notes**: Only free VPN with no data limit and no ads; NetShield ad-blocker included; Tor-enabled servers

## IVPN

IVPN is a privacy-focused VPN with strict no-logging policy, transparent ownership, and regular security audits.

- **License**: Open Source (audited, open-source VPN apps)
- **Ads**: No ads
- **Pricing**: Standard plan $6/month or $60/year; Pro plan $10/month or $100/year; team pricing available
- **Notes**: Multi-hop connections; AntiTracker feature; anonymous signup with no email required; 30-day money-back guarantee

# Operating Systems

## Tails (The Amnesic Incognito Live System)

Tails is a live operating system designed for anonymity.
It routes all connections through Tor and leaves no trace on the computer after shutdown.

- **License**: Open Source (GNU/GPL v3)
- **Ads**: No ads
- **Pricing**: Completely free; can donate to support development
- **Notes**: Fits on USB stick; used by activists, journalists, domestic violence survivors; amnesic feature leaves no trace

## Qubes OS

Qubes OS is a security-oriented OS using virtualization to isolate different tasks into separate compartments (qubes).
It minimizes attack surface through isolation.

- **License**: Open Source (free and open source)
- **Ads**: No ads (non-profit project)
- **Pricing**: Completely free; accepts donations
- **Notes**: Used by SecureDrop for journalist workstations; supports Linux and Windows virtual environments; runs on dedicated hardware

## Whonix

Whonix consists of two virtual machines: Gateway (routes all traffic through Tor) and Workstation (user applications).
It provides strong isolation and anonymity.

- **License**: Open Source (Freedom Software license)
- **Ads**: No ads
- **Pricing**: Completely free; optional donations
- **Notes**: Can be run inside VirtualBox, KVM, or Qubes OS; designed to prevent IP address leaks; 100% open source

## GrapheneOS

GrapheneOS is a privacy and security-focused mobile OS for Pixel devices.
It hardens Android with enhanced sandboxing, permission controls, and exploit mitigations.

- **License**: Open Source (non-profit open source project)
- **Ads**: No ads
- **Pricing**: Completely free
- **Notes**: Based on Android Open Source Project (AOSP); removes Google Play Services dependency; compatible with Android apps; no profit-based model

## PureOS

PureOS is a Debian-based GNU/Linux distribution focusing on privacy, security, and freedom.
It is endorsed by the Free Software Foundation and uses exclusively free software.

- **License**: Open Source (fully free and open source)
- **Ads**: No ads
- **Pricing**: Free; optional subscription added in 2024 to fund development
- **Notes**: Default OS on Librem devices; no proprietary software or blobs; privacy features enabled by default (DuckDuckGo search, HTTPS Everywhere)

# Password Managers

## Bitwarden

Bitwarden is an open-source password manager with end-to-end encryption.
It offers self-hosting options and is widely trusted for personal and business use.

- **License**: Open Source (source code on GitHub)
- **Ads**: No ads
- **Pricing**: Free tier available (unlimited passwords, unlimited devices); Premium $10/year; Family $40/year; Business plans available
- **Notes**: Self-hosting option available (Bitwarden_RS); cross-platform; audited by third parties

## KeePassXC

KeePassXC is an offline password manager with strong encryption.
It stores passwords in a local encrypted database with no cloud dependency.

- **License**: Open Source (GNU GPL)
- **Ads**: No ads
- **Pricing**: Completely free; accepts donations
- **Notes**: Cross-platform (Windows, macOS, Linux); no internet connection required; supports hardware keys; actively maintained fork of KeePassX

## Proton Pass

Proton Pass is an end-to-end encrypted password manager integrated with the Proton ecosystem.
It supports hide-my-email aliases for account creation.

- **License**: Open Source (open source and audited)
- **Ads**: No ads
- **Pricing**: Free tier available (unlimited logins, unlimited passwords, 2 vaults); Plus plan $3.99/month or $35.88/year; included in Proton Unlimited
- **Notes**: End-to-end encryption; integrated with Proton Mail; passkey support; email alias generation

# Cloud Storage

## Nextcloud

Nextcloud is a self-hosted cloud storage solution with end-to-end encryption.
Users maintain full control over data with no third-party access.

- **License**: Open Source (AGPL license)
- **Ads**: No ads
- **Pricing**: Free for self-hosting; Enterprise subscriptions available with pricing upon request
- **Notes**: Self-hosted only (no official hosted service); extensive app ecosystem; full data sovereignty; can be integrated with existing storage

## Cryptomator

Cryptomator is client-side encryption tool for cloud storage.
It works with any cloud provider by encrypting files locally before upload.

- **License**: Open Source (free and open source)
- **Ads**: No ads
- **Pricing**: Free for desktop (Windows, macOS, Linux); mobile apps require one-time purchase ($12.99 for iOS full version); Cryptomator Hub for teams has subscription pricing
- **Notes**: AES-256 encryption; works with Dropbox, Google Drive, OneDrive; individual file encryption

## Proton Drive

Proton Drive is encrypted cloud storage with zero-access encryption.
No file metadata or content is accessible to the service provider.

- **License**: Open Source
- **Ads**: No ads
- **Pricing**: Free tier available (5 GB storage); paid plans from approximately $4-12/month; included in Proton Unlimited
- **Notes**: End-to-end encryption; based in Switzerland; available on all platforms; no third-party access

# Privacy Extensions and Tools

## uBlock Origin

uBlock Origin is a free, open-source browser extension for content filtering and ad blocking.
It is highly efficient with low CPU and memory usage.

- **License**: Open Source (GPLv3)
- **Ads**: Blocks ads (does not show ads)
- **Pricing**: Completely free
- **Notes**: Available for Firefox, Chrome, Edge; highly customizable filter lists; prevents ads from loading entirely rather than just hiding them

## Privacy Badger

Privacy Badger is a browser extension from EFF that automatically learns to block invisible trackers using heuristic-based approach.

- **License**: Open Source (GPL)
- **Ads**: Blocks trackers (not specifically an ad blocker)
- **Pricing**: Completely free
- **Notes**: Developed by Electronic Frontier Foundation; sends Do Not Track and Global Privacy Control signals; no manual configuration required

## LocalCDN

LocalCDN is a browser extension that emulates Content Delivery Networks locally.
It prevents third-party requests to external CDNs that could track users.

- **License**: Open Source
- **Ads**: No ads
- **Pricing**: Completely free; accepts donations
- **Notes**: Pre-configured and customizable; contains frameworks/libraries locally (10 MB size); prevents CDN tracking

## LibreOffice

LibreOffice is an open-source office suite alternative to Microsoft Office.
It requires no cloud connectivity and collects no data.

- **License**: Open Source (MPL 2.0)
- **Ads**: No ads
- **Pricing**: Completely free
- **Notes**: Compatible with Microsoft Office formats; successor to OpenOffice; backed by non-profit Document Foundation; no forced subscription

## Jitsi Meet

Jitsi Meet is an open-source video conferencing solution requiring no account for meetings.
It provides end-to-end encryption for one-to-one calls.

- **License**: Open Source (WebRTC based)
- **Ads**: No ads
- **Pricing**: Completely free for self-hosting; hosted version free at meet.jit.si; enterprise support available
- **Notes**: Self-hosted option available; no registration required; supports up to 250 participants with recording options

## Syncthing

Syncthing is a decentralized file synchronization tool with P2P architecture.
It has no central server and uses end-to-end encrypted transfers.

- **License**: Open Source (MPL 2.0)
- **Ads**: No ads
- **Pricing**: Completely free; accepts donations
- **Notes**: Available for Windows, macOS, Linux, Android, BSD; real-time synchronization; no cloud storage required; iOS client available as paid app (iSyncthing)
