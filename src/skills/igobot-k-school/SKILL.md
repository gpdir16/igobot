# 학교 급식 및 시간표 조회 스킬

## 공통: 학교 이름 파악

**우선순위 순서대로** 학교 이름을 파악한다:

1. **사용자 메시지에 학교 이름이 포함된 경우** → 그 이름으로 바로 조회한다. 즉시 진행, 절대 다시 묻지 않는다.
2. **`memory_list`로 확인한 저장된 학교 정보** → 저장된 이름으로 진행한다.
3. 위 둘 다 없는 경우에만 사용자에게 학교 이름을 질문한다.

학교 이름을 새로 파악했으면 저장한다:
```
memory_save(name="school-info", content="# 학교 정보\n\n학교명: OO고등학교\n학년: X학년\n반: X반")
```

검색 결과 동명이교가 여러 개 나올 때만 지역을 추가로 질문한다.

날짜가 메시지에 포함된 경우(예: "20250307", "2025-03-07", "내일", "어제") → 해당 날짜로 조회한다. 없으면 오늘.
오늘 급식 정보가 없는 경우 다음 급식 정보를 조회하고 사용자에게 알린다.

## 급식 조회 — 학교알리미 (schoolinfo.go.kr)

### Step 1: 학교 검색 페이지 접속

`browser_fetch` 또는 `browser_interact`로 아래 학교알리미 검색 URL에 접속한다:

```
https://www.schoolinfo.go.kr/ei/ss/Pneiss_a01_s0.do
```

검색창에 학교명을 입력하고 검색 버튼을 클릭한다.

### Step 2: 학교 선택

검색 결과 목록에서 정확한 학교를 선택한다. 동명이교가 있을 경우 사용자에게 지역을 물어본다.

학교 선택 후 학교 상세 페이지로 이동하면, 상단 탭에서 **"급식"** 또는 **"학교급식"** 탭을 찾아 클릭한다.

### Step 3: 날짜 선택 및 메뉴 추출

- 오늘 날짜이면 기본 표시된 급식을 확인
- 다른 날짜이면 날짜 선택기에서 해당 날짜로 이동
- 조식/중식/석식 메뉴를 모두 추출
- 알레르기 번호(괄호 안 숫자)는 제거하고 메뉴명만 정리
- 칼로리 정보가 있으면 함께 표시

## 시간표 조회

### Step 1: 컴시간 사용 여부 확인

`memory_list`로 저장된 정보를 확인한다.

- **컴시간 코드가 저장된 경우** → Step 3으로 바로 이동
- **"미사용"으로 저장된 경우** → 시간표를 가져올수 없으므로 사용자에게 알림
- **저장 없음** → Step 2 진행

### Step 2: 컴시간알리미에서 학교 검색

`browser_interact`로 컴시간 사이트에 접속한다:

```
http://컴시간학생.kr/ (구시대적 웹사이트라 https를 지원하지 않음)
```

학교명으로 검색한다.

- **학교가 검색되면**: 해당 학교의 고유 코드(URL의 파라미터 또는 학교 ID)를 추출하여 저장:
  ```
  memory_save(name="comcigan-code", content="# 컴시간 학교 코드\n\n코드: XXXXX")
  ```
  → Step 3으로 이동

- **학교가 검색되지 않으면**: 미사용 표시 후 사용자에게 알림:
  ```
  memory_save(name="comcigan-code", content="# 컴시간 학교 코드\n\n상태: 미사용")
  ```

### Step 3: 컴시간 시간표 조회

저장된 학교 코드로 컴시간 시간표 페이지에 접속한다.

학년/반은 다음 우선순위로 파악한다:
1. 사용자 메시지에 포함된 경우 (예: "2학년 3반") → 바로 사용
2. `memory_list`로 확인한 저장된 학교 정보 → 저장값 사용
3. 위 둘 다 없을 때만 질문한다

해당 날짜 요일의 시간표를 추출한다.
