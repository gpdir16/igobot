# 사용자가 갤러리 이름이나 글을 명시했을때

1. 브라우저 제어 도구를 이용하여 DuckDuckGo에서 "디씨인사이드 {갤러리네임} {글제목-제공된 경우}" 검색
2. 검색 결과에서 해당 글을 찾아 클릭하여 게시글 페이지로 이동
3. 게시글 내용과 댓글을 크롤링하여 사용자에게 제공

## 사용자가 최근 글 요약을 요청했을때

기본적으로 위의 지침을 따르되 명시적인 기간이 포함되지 않은 경우 기본적으로 3일의 요약본을 제공한다.
